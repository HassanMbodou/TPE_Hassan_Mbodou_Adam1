# Guide de Contribution

Merci de votre intérêt pour contribuer à ce projet e-commerce !

## Comment Contribuer

### Signaler un Bug
1. Vérifiez que le bug n'a pas déjà été signalé
2. Ouvrez une issue avec :
   - Description claire du problème
   - Étapes pour reproduire
   - Environnement (OS, version PHP, etc.)
   - Captures d'écran si pertinent

### Proposer une Amélioration
1. Ouvrez une issue décrivant l'amélioration souhaitée
2. Expliquez pourquoi cette amélioration serait utile

### Soumettre du Code
1. Forkez le repository
2. Créez une branche pour votre fonctionnalité (`git checkout -b feature/nouvelle-fonction`)
3. Commitez vos changements (`git commit -m 'Ajout de la nouvelle fonctionnalité'`)
4. Pushez vers la branche (`git push origin feature/nouvelle-fonction`)
5. Ouvrez une Pull Request

## Standards de Code

### PHP
- Utilisez des noms de variables et fonctions en anglais
- Commentez le code complexe
- Respectez PSR-12 si possible

### HTML/CSS
- Utilisez des classes sémantiques
- Rendez le code responsive

### Git
- Utilisez des messages de commit descriptifs
- Commitez fréquemment avec des changements logiques

## Tests
- Testez toutes les fonctionnalités avant de soumettre
- Vérifiez que l'installation fonctionne sur un environnement propre

## Contact
Pour toute question, contactez l'auteur du projet.
