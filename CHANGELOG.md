# Changelog

Tous les changements notables apportés à ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
et ce projet respecte [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-08

### Ajouté
- Site e-commerce complet en PHP/MySQL
- Système d'inscription et connexion utilisateurs
- Catalogue de produits avec panier d'achat
- Validation de commandes
- Panel d'administration
- Interface responsive avec CSS
- Scripts JavaScript de base
- Configuration Docker
- Intégration continue avec GitHub Actions
- Documentation complète (README, CONTRIBUTING)
- Fichiers de configuration (.gitignore, .editorconfig, .gitattributes)

### Fonctionnalités
- Gestion des utilisateurs avec hashage des mots de passe
- Gestion du panier avec sessions
- Mise à jour automatique du stock
- Interface d'administration pour les produits et commandes

### Sécurité
- Protection contre les injections SQL avec PDO
- Hashage des mots de passe
- Sessions PHP sécurisées

### Développement
- Structure MVC simplifiée
- Code commenté et organisé
- Tests de syntaxe PHP automatisés
