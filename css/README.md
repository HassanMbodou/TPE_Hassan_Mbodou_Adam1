# Styles CSS

Ce dossier contient les feuilles de style pour le site e-commerce.

## Fichiers

- `style.css` : Styles principaux du site (layout, responsive, formulaires, etc.)

## Structure CSS

- Reset et variables globales
- Header et navigation
- Layout principal (grid pour les produits)
- Styles des formulaires
- Responsive design
- Footer

## Technologies

- CSS3 avec Flexbox et Grid
- Design responsive (mobile-first)
- Palette de couleurs cohérente

## Personnalisation

Pour modifier l'apparence :
1. Éditez `style.css`
2. Testez sur différentes tailles d'écran
3. Vérifiez la compatibilité navigateurs
